# DPM_Operate
利用Dhizuku获取DeviceOwner权限，然后进行一些操作

## 本软件功能:
1. 限制指定/全部APP的卸载
2. 限制安装APP
3. 设置DeviceOwner锁屏提示文字
4. 设置组织名
5. 禁用截屏
6. 禁用相机
7. 禁用ADB调试
